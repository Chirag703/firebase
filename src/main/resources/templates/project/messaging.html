Cloud Messaging

Send notifications and messages to your app users across platforms.

Cloud Messaging is enabled. You can send push notifications to your users.
Cloud Messaging is not enabled. Enable it in Project Settings → Services.
Compose Notification
Enable Messaging service to send notifications.
Message Types
Notification Messages
Displayed automatically by the FCM SDK when the app is in the background.
Data Messages
Handled by your app code. Contains a key-value payload.
Combined Messages
Contains both notification and data payloads.
API Example

Send a push notification via the REST API:

POST /api/v1/project/id/messaging/send
X-API-Key: YOUR_API_KEY

{
  "title": "Hello World",
  "body": "This is a push notification",
  "topic": "all",
  "data": { "key": "value" }
}
